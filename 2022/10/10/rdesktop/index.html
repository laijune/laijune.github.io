Linux 远程桌面 Windows

Posted on 2022-10-10

上一章我们尝试了windows远程连接Linux的机器，只需要在Linux客户端上开启服务即可，这次我们尝试Linux连接Windows，还有Linux连接Linux…

Linux 远程连接 Windows 有两种方法，rdesktop 和 freerdp；

Ubuntu 安装这两种包很方便，只需 apt install rdesktop 和 apt install freerdp 即可，因为 Ubuntu 是基于 Deebain 架构的；

CentOS 安装则费劲很多，需要额外安装其它的软件包

演示 CentOS Stream 9 安装 rdesktop 的步骤

  1. 配置网络，能联通外网
  1. 从 github 上下载 rdesktop 的压缩包 https://github.com/rdesktop/rdesktop，安装
